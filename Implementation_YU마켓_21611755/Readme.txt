1. Regist
 - 아직 가입을 하지 않은 경우 회원가입을 통해서 ID를 만들어야 YU마켓의 기능을 이용할 수 있습니다.

2. Login
 - Login을 진행하여야 YU마켓의 기능을 이용할 수 있습니다. Login은 Email과 Password를 통해 진행됩니다.

3. 메인화면
 - 메인화면은 5개의 Navigation Bar가 하단에 위치하며, 해당 버튼을 클릭시 각 버튼에 대응하는 기능이 실행됩니다.
 - 홈 버튼은 판매/대여/경매 물품 list를 출력하는 화면입니다. 각 리스트에서 물품을 클릭하면 하단에 Snackbar가 나타나며, 해당 snackbar에서 거래신청 혹은 상세보기를 클릭하면 물품 상세 정보가 출력됩니다.
 - 물품 정보에서 판매자 이름을 클릭시 판매자의 정보를 출력합니다.
 - 물품 정보에서 채팅으로 거래를 누르면 판매자와의 채팅 화면이 출력되고, 채팅화면에서는 사진, 메시지 전송, 거래 신청이 가능합니다.
 - 경매 물품의 경우 입찰하기 기능이 나타나며, 해당 기능을 통해 물품 입찰이 가능합니다.
 - 별 모양 버튼을 누르면 찜목록에 추가되며, 이는 마이페이지에서 확인할 수 있습니다.

4. 검색
 - 검색 버튼을 누르면 검색어 입력, 속성 선택, 카테고리 선택 화면이 출력됩니다.
 - 검색 범위는 제목, 내용, 제목+내용 이 있습니다.
 - 검색어를 입력하고, 속성, 카테고리를 선택한 후 우측상단에 있는 돋보기 버튼을 클릭하면 검색 결과가 출력됩니다.

5. 내 주변
 - 내 주변 버튼은 사용자가 설정한 자주가는 장소를 중심으로 마커를 출력합니다.
 - 해당 마커 클릭시 물품 보러가기 버튼이 나타나며, 해당 버튼을 누르면 그 위치에서 판매 하는 물품 정보들이 출력됩니다.

6. 글쓰기
 - 글쓰기 버튼은 물품을 등록하는 버튼입니다.
 - 제목 입력, 거래타입 선택, 거래장소 선택, 카테고리 선택, 내용 입력을 진행하면 글쓰기가 가능합니다. 사진을 업로드 하지 않으면 기본 이미지가 출력되며, 사진 업로드는 1개의 사진만 가능합니다.
 - 모든 정보를 입력했다면 우측 상단의 디스켓 모양 버튼을 클릭하면 물품 등록이 완료됩니다.

7. 채팅
 - 채팅은 판매자 / 구매자와의 채팅 목록을 출력합니다.
 - 채팅목록에서 특정 채팅을 클릭하면 해당 판매자 혹은 구매자와의 채팅내역이 출력되며, 이 화면에서 거래 신청 및 수락이 가능합니다.

8. 회원정보
 - 마이페이지를 출력하는 버튼입니다.

9. 판매/대여내역, 구매/대여내역
 - 내가 사고 파는 물품들의 내역을 출력합니다.
 - 구매자가 거래 완료를 해야 판매자가 거래완료를 진행할 수 있습니다.
 - 만약 장기간 거래완료 버튼을 누르지 않는 판매자/구매자가 존재한다면, 1:1 문의를 이용해 주시기 바랍니다.

10. 후기보기
 - 작성한 후기, 받은 후기를 출력하는 기능입니다.
 - 본인이 받은 후기에 대해서는 삭제요청이 가능합니다.
 - 삭제 요청시 운영진이 확인 후 삭제를 진행합니다.

11. 찜목록 보기
 - 사용자가 찜한 물품 정보를 출력합니다.

12. 계정정보설정
 - 사용자의 계정정보를 출력합니다. 해당 기능을 통해 자주가는 건물을 설정할 수 있습니다.

13. 사용자 차단설정
 - 유저 프로필에서 차단하기 기능을 이용해서 차단한 사용자 목록이 표시됩니다.
 - 차단 해제 기능을 이용해 해당유저의 차단을 해제할 수 있습니다.

14. 앱 환경 설정
 - 알림설정 및 글꼴 설정을 변경할 수 있는 기능입니다.
 - 설정완료를 진행하지 않으면 설정이 저장되지 않습니다. 꼭 설정완료를 진행해주시기 바랍니다.

15. 고객센터
 - 공지사항, 자주묻는 질문, 1:1 문의 기능을 이용할 수 있는 페이지입니다.
 - 앱 사용시 궁금한 사항 등을 확인하기 위해서는 해당 페이지를 이용해주시기 바랍니다.

16. 로그아웃
 - 기본적으로 이 어플은 자동 로그인 기능을 지원합니다. 해당 기능을 통해서 접속을 해제하지 않으면 추 후 프로그램 실행시 로그인이 자동으로 진행됩니다.
 - 로그아웃 시 로그인 페이지로 페이지가 전환됩니다.
